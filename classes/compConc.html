Computer Concepts


This course is an introduction to computer concepts that teaches the most basic concepts of computer programming that can be taken during the fall semester. Students will start by learning how to use SCRATCH programming to make animated videos. For the remainder of the semester, students will focus on the intricacies of Google Suite which includes Google Docs, Google Sheets, Google Slides, Google Forms, and Google Sites.

Scratch Programming


A visual programing language that allows students to create their own games, animations, stories, or other interactive media. Students will design a scratch project that uses sprites to move across the screen, interact with the user, and develop programming logic. Scratch was created by a group from MIT and is able to be used for free at home or at school.

Python Programming


Computer programming languages allow us to give instructions to a computer in a language the computer understands. Just as many human-based languages exist, there are an array of computer programming languages that programmers can use to communicate with a computer. The unit on Python Programming will be focused on the basics of programming in a language beyond blocks like in SCRATCH programming. Students will learn how to make low level programs that helps develop computational thinking and problem solving skills.

Google Suite


Google Suite is used in everyday life in high school and beyond. Students will learn functional skills based around Google Docs, Google Sheets, Google Slides, Google Forms, and Google Sites. Projects in this unit will include making flyers, letters, presentations, quizzes, surveys, and websites. Students will be able to use the information gained from this unit to complete tasks in other classes beyond Computer Concepts.
