Website Development


This class takes place during the fall semester of the school year and is a requirement to complete before taking website programming during the second semester. Students who take this course will explore the intricacies of creating websites using HTML and CSS. The class begins by making small websites that contain recipes, birthday party invites, favorite movies, and more. By the end of the semester, students will be creating their own websites that revolve around making resumes for different celebrities and athletes.

HTML


Hypertext Markup Language (HTML) is the language which a computer translates code for the website into the visuals that users see on a web page. Students will learn how to display images, tables, hyperlinks, and more!

CSS


Cascading Style Sheets is the method in which websites are given color, positioning, and design. Students will learn how to change color of backgrounds and text, resize images, change font sizes and colors, and much more! CSS is the second step into developing websites.
