Website Programming


This class takes place during the spring semester of the school year and can only be taken after successful completion of Website Development. During this course, students will learn Javascript and how to program using a scripting language inside of Google Chrome. The process for learning Javascript will take somewhere between 8 and 10 weeks of the class. With the remainder of the course, students will create two guided websites and two websites on their own. The first guided website is a dice game that randomly rolls dice, and the second website is a website that allows the user to play drums with the keyboard. Then, on their own, students will create a website that plays rock, paper, scissors. The course culminates with a final project where students choose their final subject.

Javascript


Javascript is a scripting or programming language that allows you to implement complex features on web pages. Any time a web page changes or allows the user to interact indicates that javascript is utilized in some way. This is the third and final piece to implementing the front end (what the user sees) of a website.
